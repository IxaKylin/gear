NAME

gear-hsh - run hasher for gear repository  

SYNOPSIS

gear-hsh [options] [hasher options] [gear options]  

OPTIONS

-v, --verbose
print a message for each action;
-V, --version
print program version and exit;
-h, --help
show this text and exit.
