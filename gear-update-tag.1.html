NAME

gear-update-tag - update list of stored tags in the package repository  

SYNOPSIS

gear-update-tag [options] <name> [<value>]
gear-update-tag [options] --all [--no-clean]
gear-update-tag [options] --clean
gear-update-tag [options] --delete <name>
gear-update-tag [options] --verify  

DESCRIPTION

gear-update-tag - update list of stored tags in the package repository  

OPTIONS

-a, --all
update all tags used in rules;
-c, --clean
delete tags not used in rules;
--no-clean
do not delete tags not used in rules;
-d, --delete
delete the specified tag;
--verify
verify all tags used in rules;
-r, --rules=FILENAME
override rules file name;
-v, --verbose
print a message for each action;
-q, --quiet
do not print a message for each action;
-V, --version
print program version and exit;
-h, --help
show this text and exit.
